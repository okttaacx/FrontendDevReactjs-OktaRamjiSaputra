# Frontend Developer Technical Test - Restaurant App

Project ini dikembangkan sebagai bagian dari Technical Test posisi Frontend Developer. Aplikasi ini menampilkan daftar restoran dengan fitur filter dan halaman detail yang dilengkapi dengan peta interaktif.

## 🛠 Tech Stack & Versi
* **React.js:** ^19.2.4
* **Node.js:** v22.17.1
* **Vite:** ^8.0.4
* **Library Tambahan:**
  * `react-router-dom` (^7.14.1) - Untuk navigasi halaman
  * `react-leaflet` & `leaflet` (^1.9.4) - Untuk integrasi peta
  * `axios` (^1.15.0) - Untuk HTTP requests
