# Frontend Developer Technical Test - Restaurant App

Project ini dikembangkan sebagai bagian dari Technical Test posisi Frontend Developer. Aplikasi ini mengimplementasikan mockup UI restoran dengan fitur filter interaktif, daftar restoran, dan halaman detail yang mencakup ulasan serta peta lokasi.

## 🛠 Informasi Teknis (System Information)
* **Node.js Version:** v22.17.1
* **React.js Version:** ^19.2.4
* **Vite Version:** ^8.0.4
* **Build Tool:** npm
